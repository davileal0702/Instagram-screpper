Pesquisa de Perfil (Instaloader)

Selecione um perfil já cadastrado (sessão salva) ou use modo anônimo. O sistema gera um CSV com os dados do perfil e dos posts.

Fluxo recomendado: use Usuário já cadastrado (sessão salva). O cadastro/renovação de sessão é feito no script gerar_sessao_instaloader.py.
No modo anônimo, perfis privados não poderão ser consultados.

As sessões são lidas automaticamente de insta_sessions/.
Excluir perfil remove o arquivo de sessão salvo desta POC local.
Como cadastrar novo perfil (fácil): execute python gerar_sessao_instaloader.py no terminal e depois clique em Atualizar lista.
